# LUMIÈRE - Luxury Fashion Boutique

A premium, edge-to-edge e-commerce UI crafted with **React** and **Tailwind CSS**. Designed for high-end fashion boutiques.

## ✨ Features
- **Modern Minimalist UI**: Full-screen, luxury aesthetic.
- **Component-Based Architecture**: Modular files for easy maintenance.
- **Dynamic Views**: Seamless transitions between Collections, Customizations, and New Arrivals.
- **Responsive Design**: Optimized for all screen sizes.

## 🛠️ Tech Stack
- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Icons**: Lucide-React


## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/lumiere-boutique.git`
2. Install dependencies: `npm install`
3. Start project: `npm run dev`
